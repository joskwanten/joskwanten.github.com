Maze of Galious

This is just a very quick and dirty port of the Maze of Galious remake by braingames ( http://www.braingames.getput.com/mog/). The code is compiled with emscripten to compile C++ code to javascript and the SDL code is tweaked a little bit to make everything work again.

Source code can be found here: https://github.com/joskwanten/mog-emscripten
